# Unitale
一个基于Indextts的 AI 有声书制作工具。利用 LLM 自动拆解剧本与识别情绪，集成多角色 TTS 语音合成，支持音效(SFX)、背景音乐(BGM)混音及实时音频滤波器的自动插入和匹配，可直接在浏览器导出 WAV 成品。

Unitale 是一个轻量级的、基于AI的可视化有声书生成工作台。它旨在简化从“小说文本”到“有声作品”的转化流程。

## 核心功能:

AI 智能剧本分析: LLM自动将小说原文拆分为台词与旁白，智能识别角色、分析情绪强度，并自动建议音效与背景音乐的插入位置。

多角色语音合成: 支持对接IndexTTS 2，允许为不同角色绑定独立的音色参考音频，实现生动的多角色演绎。

## 可视化自动化音频编排:

音效 (SFX): 支持自动在台词的具体位置（如：说完 50% 时）插入音效，支持多音效叠加。

背景音乐 (BGM): 支持AI自动切换控制BGM。

音频滤波器 (Filters): 内置 Web Audio API 滤波器，可为台词添加“电话音”、“水下”、“老广播”、“机械失真”等环境特效。可为配音自动匹配适合的滤波器。

实时混音与预览: 基于 Web Audio API 的实时混音引擎，支持单句试听和全篇顺序播放（自动处理停顿、音效与 BGM 的同步）。

## 本地化与导出:

支持导出/导入工程文件 (.json)，保存所有编辑状态。

支持前端离线渲染，直接导出最终合成的 WAV 音频文件。

高度可配置: 自定义 Prompt 模板、情绪预设、音色库和音效素材库。
